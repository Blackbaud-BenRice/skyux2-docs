The split view component allows users to work through a list of items, while displaying each item's details in context. The split view component consists of two panels: the list panel, and the workspace panel. The SkySplitViewService allows consumers to control what is displayed in the workspace panel. The SkySplitViewInstance class watches for and triggers events that come from the split view component.

Specifies the default width of the left-side list panel. This property accepts number values. (Default: 320) Allows users to resize the panels by clicking and dragging the panel separator. (Default: true)
